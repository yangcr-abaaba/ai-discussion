核心功能

不再来回切标签页、复制粘贴。统一发问、自动转交、互相辩论——一个屏幕，统筹全场。

01

一句话群发

在底部输入框敲一次字，同时送达所有已登录的 AI。从此告别「同一个问题复制粘贴五遍」。

02

自动讨论

各家答完，一键把每家观点附上讨论规则、转交给其他 AI，让它们真正「对话」，而非各说各话。

03

多栏并排

每个 AI 独占一栏，可单独刷新、最大化、随时关闭。最多同时三家，互不打扰、一览无余。

04

提示词管理

常用提示词与「自动讨论」抬头规则随手存取，一点即填；增删改随心，启用即刻生效。

如何使用

不需要 API Key，不需要任何配置——装好，就用你早已登录的 AI 账号开场。

01
入席

选 AI · 登录

在底部选 1~3 位「参会者」，用它们各自的账号登录。产品不保存你的任何账号信息。

02
发问

一句话群发

输入问题、点「发送」，同一句话同时抵达所有 AI。发送方式可在 Enter / Ctrl+Enter 间切换。

03
交锋

点「自动讨论」

各家答完，一键把所有观点加上讨论规则、转交彼此，看它们互相补充、反驳、层层深入。

底层原理

基于 Electron，用隔离分区的内嵌网页直接驱动各 AI 官网——你用的，就是它们真实、完整的网页能力。

  • 直接操作真实网页

    定位输入框 → 填字 → 触发发送 → 读取回答 → 转交。全程在你已登录的官网上完成，零额外费用、零 API 密钥。

  • 为什么不是 iframe

    多数 AI 站点用 X-Frame-Options / 同源策略 / 第三方 Cookie 设防，纯 iframe 既嵌不进也操作不了；webview 才能以「第一方顶层页面」的身份绕过。
